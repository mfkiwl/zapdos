ElementL2FunctorError

The ElementL2FunctorError computes the Euclidean distance between an automatic differentiation (AD) functor representing an approximate solution and an AD functor representing an analytical exact solution. This Postprocessor is very useful for verifying the proper function of the framework through the Method of Manufactured Solutions. This class is a generalization to the functor system of ElementL2Error.

Input Parameters

  • approximateThe approximate functor. This functor has to be an ADFunctor, like a variable or an ADFunction

    C++ Type:MooseFunctorName

    Controllable:No

    Description:The approximate functor. This functor has to be an ADFunctor, like a variable or an ADFunction

  • exactThe analytic solution to compare against

    C++ Type:MooseFunctorName

    Controllable:No

    Description:The analytic solution to compare against

    Default:TIMESTEP_END

    C++ Type:ExecFlagEnum

    Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM, TRANSFER

    Controllable:No

    Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, MULTIAPP_FIXED_POINT_END, MULTIAPP_FIXED_POINT_BEGIN, FINAL, CUSTOM.
