MeshMetaDataReporter

A Reporter object for outputting mesh metadata in a human-readable form. By default, it executes only on initial.

This object is a specialization of RestartableDataReporter, which you can see for a more detailed parameter description.

See MeshMetaDataInterface for more information on the use of mesh metadata.

Example Input Syntax

The following input file snippet demonstrates the use of the MeshMetaDataReporter object.

[Reporters]
  [metadata]
    type = MeshMetaDataReporter
  []
[]
(moose/test/tests/reporters/mesh_meta_data_reporter/mesh_meta_data_reporter.i)

Advanced Parameters

(moose/test/tests/reporters/mesh_meta_data_reporter/mesh_meta_data_reporter.i)

[Mesh]
  [gmg]
    type = GeneratedMeshGenerator
    dim = 1
    nx = 1
  []
  [metadata]
    type = AddMetaDataGenerator
    input = gmg
    uint_vector_metadata_names = 'foo'
    uint_vector_metadata_values = '1 2 3 4'
  []
[]

[Problem]
  solve = false
[]

[Executioner]
  type = Steady
[]

[Reporters/metadata]
  type = MeshMetaDataReporter
[]

[Outputs]
  [out]
    type = JSON
    execute_on = initial
    execute_system_information_on = none
  []
