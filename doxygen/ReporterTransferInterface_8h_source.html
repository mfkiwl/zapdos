18/*
19 * This transfer serves as a base class for transferring reporter values between
20 * multiapps. This includes reporters, vector postprocessors, and postprocessors.
21 *
22 * The underlying purpose fo this class is to avoid using non-const access to ReporterData
23 * through FEProblemBase to set reporter data. Instead, we simply have this class
24 * as a friend to ReporterData::WriteKey and provide these protected functions for derived
25 * classes. This avoids any old object modifying reporter data.
