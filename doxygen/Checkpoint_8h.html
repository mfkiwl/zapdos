enum AutosaveType : unsigned short

Shortcut for determining what type of autosave this checkpoint is.

NONE: Not an autosave checkpoint MODIFIED_EXISTING: We took an existing checkpoint and enabled autosave checks on it. SYSTEM_AUTOSAVE: These checkpoints run in the background and output only when sent a signal.
